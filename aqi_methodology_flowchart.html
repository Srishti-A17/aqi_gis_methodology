Spatial AQI Analysis Methodology

New York State (2020–2025) | Multi-Layer Spatial Aggregation Framework

INPUT / PREPROCESSING
Raw AQI Monitoring Data
Point-level AQI measurements from monitoring stations across NYS (2020–2025)
Spatial Boundary Definition
New York State boundary shapefile used to constrain analysis extent
Dataset Integration
Merging yearly pollutant datasets into unified feature classes
Summary Statistics
Compute maximum AQI per monitoring site per year
Joined AQI Point Dataset
Summary tables joined back to AQI point features
LAYER 1: Continuous Surface Analysis (IDW)
"Capturing spatial gradients and hotspots"
IDW Interpolation
Inverse Distance Weighted interpolation applied to AQI point data to estimate continuous spatial variation
Raster Processing
  • Clip interpolated raster to NY State boundary
  • Convert raster to points for downstream analysis
Purpose: Estimate continuous spatial variation of maximum AQI across NYS, revealing gradients and pollution hotspots
LAYER 2: Hexagonal Aggregation
"Standardizing spatial units & reducing edge bias"
Hexagonal Tessellation
Creation of uniform hexagonal grid covering New York State
Spatial Join
Join AQI point data to hexagonal features
Aggregated Hexagon Values
AQI values aggregated per hexagon for uniform comparison
Purpose: Reduce point-level noise and enable standardized spatial comparison across uniform geographic units
LAYER 3: Buffer-Based Proximity Analysis
"Assessing localized influence zones"
Buffer Zone Generation
Create distance-based buffer zones around AQI monitoring stations
Proximity Exposure Patterns
Comparison of AQI behavior within increasing distance bands
Purpose: Examine spatial influence and proximity-based exposure patterns around monitoring locations
LAYER 4: Voronoi (Thiessen) Polygon Analysis
"Population-normalized exposure analysis"
Voronoi Polygon Generation
Create Voronoi polygons from AQI monitoring stations, clipped to NY State boundary
Population Data Integration
Join population data to each Voronoi zone
AQI Per Million Population
Calculate pollution exposure normalized by population density
Purpose: Normalize pollution exposure by population to identify areas with disproportionate environmental health burden
